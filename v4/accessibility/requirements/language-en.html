Language

Related to success criteria

  • 3.1.1 Language of Page
  • 3.1.2 Language of Parts

When coding web pages, specify the natural human language of content by setting a:

  • Default language using the lang attribute in the <html> element

  • Change of language using the lang atrribute in any element enclosing the text in a different language.

When using the lang attribute, use:

  • en for English
  • fr for French
  • These codes for other languages (see 639-1 column)
